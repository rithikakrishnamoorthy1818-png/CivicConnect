# CivicConnect 🚀

## 📌 Problem
Managing public washroom cleanliness and safety is difficult.

## 💡 Solution
CivicConnect is an AI-powered IoT dashboard that monitors:
- Cleanliness
- Safety (camera detection, alerts)
- Hygiene supplies (napkin availability)

## ⚙️ Features
- Real-time monitoring
- Alert system
- Multi-washroom tracking
- AI detection module

## 🛠 Tech Stack
- Python (Backend)
- HTML, CSS (Frontend)
- IoT Sensors

## ▶️ How to Run
1. Clone repo
2. Install dependencies
3. Run backend
4. Open frontend
